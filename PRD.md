# Cue Product Overview

This file is a plain-language summary of what Cue is, what the public release includes, and what is still on the roadmap.

## What Cue Is

Cue is an image-first desktop app for editing, reviewing, and exporting graphics. The app is built around a canvas, direct actions, reusable tools, and reproducible receipts.

## Who It Is For

- people who want to work from images instead of complex design software
- developers and creators who want faster visual iteration
- small teams that want repeatable edit and export workflows

## What Ships Today

- one shared desktop window with multiple session tabs
- image import and canvas editing
- design review suggestions with in-place apply
- reusable tool creation inside the app
- export with receipts

## Current Public Scope

- strongest public verification: macOS
- public release format: source plus a macOS DMG
- current export formats: PSD, PNG, JPG, WEBP, and TIFF
- current export fidelity: flattened output

## Roadmap

- public Windows and Linux verification
- richer editable export, including `.ai` and `.fig`
- fewer legacy internal names left over from earlier private versions
- stronger cross-platform release parity

## Product Rules That Still Matter

- keep the visible editing loop centered on images and direct manipulation
- keep model-backed work reproducible through receipts
- keep user-facing naming on Cue
- describe platform support honestly
