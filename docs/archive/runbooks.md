# Archived Runbooks

The original multi-agent launch, design-wave, and polish runbooks were created for private execution waves and included machine-specific paths, branch names, and temporary operating notes.

They were removed from the public repo surface during open-sourcing.

What remains true from those runbooks:

- implementation work used dedicated worktrees
- merge order mattered when shell, tool runtime, and export touched the same contracts
- product docs needed to move with code changes

For the current public workflow, use [AGENTS.md](../../AGENTS.md) and [CONTRIBUTING.md](../../CONTRIBUTING.md).
