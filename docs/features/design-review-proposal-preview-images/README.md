# Design Review Proposal Preview Images

This feature adds preview images back to design review without slowing down the moment when proposals first appear.

## User-Facing Behavior

1. `Design Review` opens the proposal tray quickly.
2. Text proposals appear first.
3. Preview images load afterward in the background.
4. The selected proposal can receive a higher-quality preview.
5. Preview failures do not block selection or apply.

## Rule

Preview media should improve confidence, not block the review flow.
